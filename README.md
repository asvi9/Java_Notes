# 📖 Course Overview

Java Programming is a fundamental course designed to introduce students to the concepts of **Object-Oriented Programming (OOP)** and modern software development practices. The course provides comprehensive knowledge of Java programming language, covering everything from basic syntax and programming constructs to advanced topics such as multithreading, collections framework, exception handling, file processing, and modern Java APIs.

The course emphasizes practical implementation of programming concepts and equips students with the skills required to develop robust, scalable, and platform-independent applications.

---

# 🎯 Course Objectives

- Understand the principles of Object-Oriented Programming.
- Learn Java programming fundamentals and syntax.
- Develop problem-solving skills using Java.
- Design reusable and maintainable software systems.
- Implement inheritance, polymorphism, abstraction, and encapsulation.
- Handle exceptions effectively to build reliable applications.
- Utilize Java Collections Framework for efficient data management.
- Implement multithreading and concurrent programming concepts.
- Work with file handling and data serialization techniques.
- Explore modern Java features including Lambda Expressions and Stream API.

# 📚 Course Modules & Weightage

| Unit | Module Name | Topics Covered | Hours | Weightage |
|------|-------------|----------------|--------|------------|
| **1** | **Overview of OOP and Java** | Programming Paradigms, Procedural vs OOP, Features of OOP, History of Java, JVM Architecture, JDK, JRE, Java Program Structure, Compilation & Execution Process, Environment Setup, IDEs | **4** | **10%** |
| **2** | **Java Language Basics** | Variables, Data Types, Type Casting, Wrapper Classes, Autoboxing, Operators, Conditional Statements, Loops, Break & Continue Statements | **7** | **15%** |
| **3** | **Arrays and Strings** | One-Dimensional Arrays, Multi-Dimensional Arrays, Array Operations, String Class, StringBuffer, StringBuilder, String Manipulation Methods | **6** | **17%** |
| **4** | **Object-Oriented Programming (OOP)** | Classes, Objects, Constructors, Access Modifiers, Encapsulation, Inheritance, Polymorphism, Abstraction, Interfaces, Functional Interfaces, this, super, final Keywords | **12** | **30%** |
| **5** | **Exception Handling** | Errors & Exceptions, Checked & Unchecked Exceptions, try-catch, finally, throw, throws, Custom Exceptions, Best Practices | **6** | **15%** |
| **6** | **Collections Framework** | List Interface, ArrayList, LinkedList, Set Interface, HashSet, TreeSet, Map Interface, HashMap, TreeMap, Iterators, Generics | **5** | **10%** |
| **7** | **Advanced Java Foundations** | Multithreading, Synchronization, File Handling, Streams, Serialization, Deserialization, Lambda Expressions, Stream API, Optional Class | **5** | **8%** |
